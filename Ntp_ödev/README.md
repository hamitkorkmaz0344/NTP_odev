# Ntp Ödev - C# Projeleri

Bu depo, ayrı klasörlerde yer alan 7 adet .NET 8.0 konsol projesi içerir.

## Projeler

| Klasör | Açıklama |
|--------|----------|
| chat uygulaması | Konsol tabanlı sohbet (mesaj gönder/listele) |
| Çoklu dil çeviri | TR/EN/DE sözlük çevirisi |
| LLM chatbot | Basit kural tabanlı chatbot |
| MI_math | Matematik işlemleri (hesap makinesi) |
| ram tabanlı | Bellekte anahtar-değer deposu |
| sesliAsistan | Metin komutları (saat, tarih, vb.) |
| Yüztanıma | Yüz tanıma simülasyonu (ID eşleştirme) |

## Çalıştırma

Her proje için ilgili klasöre girip:

```bash
dotnet run
```

Derlemek için:

```bash
dotnet build
```
